Not your first neoan3 project? We've got some news.

Since 2016 we have kept changes non-breaking. However, we are excited to now introduce version 3 to you!

What's new?

Unicore Singleton

With the introduction to modular rendering the Unicore singleton is more flexible than ever.

Models

Working with databases much? The new version will blow your mind!

Tests

Make sure you have the newest cli version to get testing done in no time.

Providers

Introduced in the LTS of neoan3 v2.x, providers have become a major part of decoupling your application while remaining fully flexible.

CLI

Worry not, the cli can handle your old neoan3 projects going forward. We introduced a version-hook that will allow flexible behavior across versions with the same cli helper.
